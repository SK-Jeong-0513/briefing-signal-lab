# 주간 기술 브리핑 메일러 (Google Apps Script)

무료 구독자에게 주간 기술 브리핑(무료 부분)을 이메일로 발송하고, 이메일 링크로 선호도/수신거부를 반영하는 샘플 메일러.

- **경로:** Google Apps Script. GitHub Pages 사이트와 독립 — 사이트가 어디에 호스팅되든 무관.
- **발송:** `sendWeekly()` 수동 실행.
- **위치:** 이 폴더는 배포되지 않음(사이트는 `public/`만 서빙). 버전관리·복사용 원본.

## 2-시트 모델

| 시트 탭 | 역할 | 발송기 |
|---|---|---|
| **설문지 응답 시트1** (`RESP_SHEET`) | 신원·동의 원천. 최초 1회 가입 기록. | **읽기 전용** (`이메일 주소`, `메일 수신 동의`, `관심 키워드`) |
| **관심분야(기술)** (`PREF_SHEET`) | 구독 중 선호도 상태. | **읽기/쓰기** — 헤더 자동 생성, 이메일 링크로 갱신 |

`관심분야(기술)` 스키마(자동 생성): `이메일 | 관심 분야 | 상태 | 갱신`
- 관심 분야 = 콤마구분 가동 분야 라벨(`AI 인프라, 반도체 공급망`). 이메일 토글 링크가 갱신.
- 상태 = `구독` / `수신거부`.
- (경제·금융 브리핑은 `관심분야(경제)`/`관심분야(금융)` 탭에 같은 구조로.)

## 설치 (약 10분)

1. **응답 구글 시트**(여러 탭 포함) 열기 → **확장 프로그램(Extensions) → Apps Script**.
   - ⚠️ 반드시 **시트에서** 열 것(폼 아님). `script.google.com` 독립 프로젝트면 `CFG.SHEET_ID`에 시트 URL의 `/d/<ID>/edit` ID를 넣으면 동작.
2. `Code.gs` 전체 붙여넣기.
3. **CFG** 확인:
   - `SALT` → 아무 문자열로 교체.
   - `RESP_SHEET`/`PREF_SHEET` → 탭 이름과 정확히 일치(`설문지 응답 시트1` / `관심분야(기술)`).
   - `RESP_COL`(`이메일 주소`/`메일 수신 동의`/`관심 키워드`) → 응답 시트 헤더와 일치.
   - `CONSENT_TRUE_INCLUDES` → 수신 동의 셀 값에 포함된 문자열(기본 `동의`).
4. **배포 → 새 배포 → 유형: 웹 앱** / 실행: 나 / 액세스: **모든 사용자** → 배포. 웹 앱 URL(`.../exec`)을 `CFG.WEBAPP_URL`에 붙여넣고 저장.
   - (웹앱 없으면 수신거부는 mailto, 분야 토글 링크는 숨김.)
5. 함수 `sendWeekly` 선택 → 실행 → **최초 1회 권한 승인**.
   - `CFG.TEST_MODE = true`(기본)면 **나(OPERATOR_EMAIL)에게만 1통** → 미리보기.
6. OK → `CFG.TEST_MODE = false` → 다시 실행 → **수신동의자 전원 발송**.

## 발송 로직

- 응답 시트에서 `이메일 주소` + `메일 수신 동의`(O) 필터. 중복 이메일은 1건만.
- 각 구독자를 `관심분야(기술)`에서 이메일로 조회:
  - **있으면** → 그 분야만. `상태=수신거부`면 스킵.
  - **없으면(최초)** → 가동 분야 전체 발송 + `관심분야(기술)`에 행 자동 생성(seed, 기본=전체).
- 이메일의 **분야 토글/수신거부 링크** 클릭 → `doGet` 웹앱이 `관심분야(기술)`에 upsert.
- `관심 키워드`는 메일 상단에 개인화 표시(발송 필터엔 미사용).

## 매주 발송

- `ISSUE` 객체만 그 주 내용으로 교체(무료 부분 = 사이트 `public/assets/content/tech.js`의 signals/headliner.summary와 동일).
- `sendWeekly` 실행. (수동 트리거 = 사람이 발행 스위치)

## 개인정보/발송 규칙

- 이메일(개인정보)은 시트 밖 유출 금지. 링크에는 이메일 대신 **해시 토큰**만.
- 수신거부 링크 필수(정보통신망법). 클릭 시 `관심분야(기술)`의 `상태`를 `수신거부`로 → 다음 발송 제외. (응답 시트는 안 건드림)
- 발송 한도: GmailApp 무료 ~100통/일. 지인 소수 샘플엔 충분. 대량·정식은 도메인 인증 ESP(Resend/SES 등)로 이관.

## 정식 이관 시(향후)

커스텀 도메인 + 백엔드로 가면: SPF/DKIM/DMARC 건 전용 ESP 발송, 선호도·구독자는 실제 DB/API로. 이 Apps Script는 그때 대체되는 임시 샘플.
